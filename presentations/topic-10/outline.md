# Topic 10: Alternative Tooling: Mabl (Low-Code)
## Learning Objectives
By the end of this session, participants will be able to:
- Understand low-code/no-code test automation
- Evaluate when Mabl or similar tools fit the organization's needs
- Compare scripted vs low-code approaches

### Presentation Outline (15-20 minutes)

1.  **Introduction: Low-Code Testing (3 min)**
    - What is low-code automation?
    - Market trends in test automation
    - When to consider low-code tools

2.  **Mabl Platform Overview (5 min)**
    - Unified test automation platform
    - AI-powered features (auto-healing, GenAI)
    - Web, mobile, API, accessibility testing
    - Cloud-native architecture

3.  **Mabl vs Scripted Frameworks (5 min)**
    - Speed of test creation
    - Maintenance burden comparison
    - Skill requirements
    - Cost analysis
    - Flexibility trade-offs

4.  **Demo: Mabl Interface (5 min)**
    - Show Mabl test creation workflow
    - Highlight AI-assisted features
    - Discuss when to choose Mabl

5.  **Q&A / Wrap-Up (2 min)**

### Demo Script

**Objective:** Show Mabl's low-code approach and AI features.

**Content (Screenshots/Video):**
1.  **Test Creation Interface:**
    - Show the visual test recorder
    - Demonstrate click-and-select actions
    - No coding required for basic tests

2.  **Auto-Healing Feature:**
    - Explain how AI adapts to UI changes
    - Show before/after examples
    - 85% maintenance reduction claim

3.  **GenAI Features:**
    - Natural language to test steps
    - AI-assisted test analysis

**Comparison Discussion (3 min):**
| Aspect | Mabl | WebdriverIO |
|--------|------|-------------|
| Setup Time | Minutes | Hours/Days |
| Coding | None/Low | High |
| Maintenance | AI-assisted | Manual |
| Flexibility | Limited | Unlimited |
| Cost | Subscription | Free + Infra |
